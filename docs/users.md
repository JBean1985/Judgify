# Utilizadores do Judgify

## Introdução

O Judgify é uma plataforma única para toda a comunidade da Patinagem Artística.

Todos os utilizadores recorrem ao mesmo motor de análise, mas cada perfil visualiza a informação de forma diferente.

O objetivo é apresentar apenas a informação realmente útil para cada pessoa.

---

# Treinador

## Objetivo

Preparar atletas e otimizar programas.

## Funcionalidades

* Construção de programas
* Biblioteca de elementos
* Planeamento de competições
* Análise de vídeo
* Comparação entre treinos
* Evolução das atletas
* Sugestões da IA

## Informação apresentada

* Valor Base
* GOE
* PCS
* Regras ISU
* Validação técnica
* Estatísticas
* Recomendações

---

# Juiz

## Informação apresentada

* Regras ISU
* GOE sugerido
* Níveis
* Validação automática

# Familiar

## Objetivo

Compreender a prestação da atleta.

## Funcionalidades

* Acompanhar competições
* Ver vídeos
* Consultar pontuação prevista
* Explicação da nota
* Evolução da atleta

## Informação apresentada

Nunca deve apresentar linguagem demasiado técnica.

A aplicação deve explicar:

* O que correu bem.
* O que pode melhorar.
* Porque perdeu pontos.
* Como foi calculada a pontuação.

# Filosofia

Todos os utilizadores utilizam o mesmo motor.

O que muda é apenas a forma como a informação é apresentada.

A prioridade do Judgify é adaptar a experiência ao contexto de cada utilizador, mantendo sempre a simplicidade, a transparência e a confiança.
